#Usage Examples

## 1. View the Webpage (HTML)
1. Navigate to the project folder.
2. Locate the `index.html` file.
3. Double-click `index.html` or right-click and open it with a web browser.
4. The webpage will load and display the HTML content.



### 2. View JavaScript Console Output
1. Open `index.html` in a web browser.
2. Right-click anywhere on the page and select **Inspect**.
3. Click on the **Console** tab.
4. You will see the JavaScript message printed from `script.js`.



### 3. Modify Page Styling (CSS)
1. Open the `style.css` file in a text editor.
2. Change any CSS property (for example, background color or text color).
3. Save the file.
4. Refresh the browser page to see the updated styling.
